Deploy from GitHub

  1. Create a GitHub repo and upload this folder.
  2. In Netlify: Add new site → Import from Git → choose your repo.
  3. Build settings: Build command (empty), Publish directory /.
  4. Add env var OPENAI_API_KEY (and optional OPENAI_MODEL).
